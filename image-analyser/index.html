Unicorn Image Analyser

Load an image in to generate a pixel map and palette for use with the Unicorn pixel-art python files.
Refer to the sizes below to use with the Unicorn LED matrix you are using.

Images with an indexed palette of around 16 to 32 colours are recommended.

Visit the Github pages to get started / Alternatively create pixel art from scratch using the pixel editor.
